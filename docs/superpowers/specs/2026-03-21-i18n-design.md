# i18n (Chinese/English) Design Spec

## Overview

Add Chinese/English language switching to the personal website. A nav button toggles between languages. Default follows browser language. All visible text is translated.

## Decisions

| Decision | Choice |
|----------|--------|
| Default language | Browser language detection (zh → Chinese, else English) |
| Persistence | localStorage |
| Toggle style | Text button: shows "中文" in EN mode, "EN" in ZH mode |
| Library | None — simple hook + data objects |
| Translation scope | All visible text including project data |
| Role/brand names | Keep English when it sounds more natural (Co-Founder, Founder, etc.) |

## Files

| File | Action |
|------|--------|
| `src/hooks/useLanguage.js` | Create — language state, browser detection, localStorage |
| `src/data/i18n.js` | Create — UI text translations (en/zh) |
| `src/data/projects.js` | Modify — bilingual fields for subtitle, description, links |
| `src/components/Navbar/Navbar.jsx` | Modify — add language toggle button, accept `lang` prop |
| `src/components/Navbar/Navbar.module.css` | Modify — style for language toggle |
| `src/pages/Home/Home.jsx` | Modify — read text from i18n, pass lang down |
| `src/pages/ProjectDetail/ProjectDetail.jsx` | Modify — read bilingual project data |

## Hook: useLanguage

```js
// Initialize: localStorage → browser language → 'en'
// Browser detection: navigator.language starts with 'zh' → 'zh', else 'en'
// Returns { lang, toggleLang }
// Side effect: saves to localStorage
```

## Data: projects.js bilingual fields

- `title` — stays as string (brand names don't change)
- `subtitle` — `{ en: '...', zh: '...' }`
- `role` — stays as string (keep English for all roles)
- `description` — `{ en: '...', zh: '...' }`
- `links[].label` — `{ en: '...', zh: '...' }`
- `cardType`, `image`, `id`, `url`, `icon` — unchanged

## Navbar change

- New prop: `lang`, `onToggleLang`
- Button placed next to ThemeToggle (right side of nav)
- Shows "中文" when `lang === 'en'`, "EN" when `lang === 'zh'`
- Same styling as ThemeToggle (transparent, inherit color, hover opacity)
